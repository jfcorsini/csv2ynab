# Bank to YNAB

Convert your bank CSV exports to YNAB-compatible format. All processing happens in your browser.

## Usage

1. Upload your bank's CSV file
2. Map columns to YNAB format (Date, Payee, Memo, Inflow/Outflow)
3. Download the converted file

## Development

```bash
npm install
npm run dev
```

Built with React, TypeScript, and Vite.
